# Movie Guide 🎥🍿

Discover amazing movies with our Movie Guide! Visit: https://itsiamdev.github.io/Movie-Guide/

## Features

- 🔍 Search for movies by title
- ⭐ View IMDb ratings and details
- 🎭 Explore genres, cast, and plot summaries
- 📱 Responsive design for all devices

## How to Use

1. Enter a movie name in the search box
2. Click the "Search" button or press Enter
3. Enjoy detailed information about your favorite movies!

## Technologies Used

- HTML5
- CSS3
- JavaScript
- OMDb API
